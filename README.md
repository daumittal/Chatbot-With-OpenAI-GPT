# Chatbot-With-OpenAI-GPT
In this project, our objective is to develop a chatbot using Flask. We'll integrate the GPT-3 API into our chatbot to generate responses to user queries effectively. Additionally, we'll implement a mechanism to store and manage a history of all user queries and the corresponding API responses.


GPT-3, the third iteration of the Generative Pre-trained Transformer (GPT) series, is a neural network model designed to generate human-like text. Trained on nearly 570 GB of text data, it boasts 175 billion parameters, requiring 800 GB of storage. The text it produces is of exceptionally high quality, often indistinguishable from human writing.

This project focuses on building a Flask-based chatbot integrated with the GPT-3 API. By leveraging the API, the chatbot can respond to user queries effectively. Additionally, the project aims to maintain a comprehensive history of all user queries and corresponding API responses.
